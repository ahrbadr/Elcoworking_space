# Setup Instructions

## Quick Installation
1. Download all project files
2. Install Python dependencies: `pip install flask`
3. Run: `python app.py`
4. Open: http://127.0.0.1:5000

## Directory Structure
Ensure you have these directories:
- `templates/` - contains index.html
- `data/` - will be created automatically for data storage

## First Run
On first run, the system will create all necessary data files automatically.
